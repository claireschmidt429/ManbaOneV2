# Manba One V2 Modder Tool

Tool to mod and customize my [Manba One V2](https://s.click.aliexpress.com/e/_oEKvolv) game controller.

## Features & Roadmap

- [x] Identify/extract firmware
- [x] Upload firmware
- [ ] Send keypresses when recording macro
- [ ] Change permanent image while using
- [ ] Change boot image
- [ ] Make the actual tool that modifies the firmware :)

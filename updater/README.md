
## Usage

1. Disconnect and turn off the controller
2. Open one of the FW Updaters
3. Hold the left stick down (LS) and turn the controller on
4. Connect the controller
5. Follow the instructions

## FW Updaters

`ManbaOne_V2.21.exe` – The original updater tool.  
`ManbaOne_V2.21_modded_to_update_already_updated.exe` – A modified version that allows re-installing the firmware even if it's already up to date:
